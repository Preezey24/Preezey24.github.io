About me

Previously working as a Project Engineer & Project Manager in large mining operations with an emphasis on problem solving and team work has successfully prepared me for a transition into the world of software development.

I particularly enjoy developing my own algorithms to assist in the implementation of various project features as well as using them to solve the problems posed by others. I readily accept the challenge of debugging code when things aren't going too smoothly and I am always eager to learn new things.
